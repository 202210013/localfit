## 1. About This Guide

This manual is a detailed, customer-focused guide for using LocalFit.

It covers complete user workflows:

- Creating and accessing your account
- Finding and ordering products
- Managing your cart
- Tracking and completing orders
- Submitting ratings and reviews
- Using realtime messaging

This guide does not include admin operations.

## 2. System Overview

LocalFit is an ecommerce platform where users can browse apparel products, order by size, track order status, confirm pickup, and communicate through messaging.

### 2.1 Core User Features

- User registration and login
- Product listing and product detail viewing
- Size-aware ordering
- Cart updates and checkout flow
- Order status tracking
- Pickup confirmation
- Rating and review submission
- Realtime chat messaging

### 2.2 Main User Pages

- /register
- /login
- /product-main
- /product-listing
- /cart
- /orders
- /messages

## 3. Account Setup and Login

### 3.1 Register a New Account

1. Open the Register page.
2. Enter required fields.
3. Submit your details.
4. Wait for successful account creation confirmation.
5. Go to Login.

Tips:

- Use a valid email and an active mobile number.
- Double-check spelling before submitting.

### 3.2 Log In to Your Account

1. Open Login page.
2. Enter your credentials.
3. Click login/sign in.
4. After successful login, access protected pages.

If login fails:

- Recheck your email and password.
- Ensure internet connection is stable.
- Retry after a few seconds.

### 3.3 Log Out Safely

1. Open menu/profile area.
2. Click Logout.
3. Confirm if prompted.
4. You are redirected to public or login page.

## 4. User Dashboard and Navigation

After login, navigate using the top menu or route links.

### 4.1 Navigation Quick Reference

- Product Main: quick landing page
- Product Listing: browse all available items
- Cart: review selected items
- Orders: track status and complete post-order actions
- Messages: communicate with support/admin

### 4.2 Protected Access Behavior

When your session expires, protected pages may redirect you to login.
This is normal security behavior.

## 5. Browsing and Searching Products

### 5.1 Open Product Listing

1. Go to Product Listing.
2. Scroll through available products.
3. Use search input to find by name or keyword.

### 5.2 Read Product Information

Each product usually contains:

- Product name
- Price
- Image
- Available sizes
- Additional details/description

### 5.3 Best Practices While Browsing

- Compare sizes before adding to cart.
- Check stock or availability hints.
- Confirm product details before placing order.

## 6. Selecting Size, Quantity, and Pickup Date

### 6.1 Size Selection

Before ordering, select your intended size.
Order actions may be blocked if size is not selected.

### 6.2 Quantity Selection

Choose a valid quantity.
If the value is invalid, update it before proceeding.

### 6.3 Pickup Date Selection

Some flows require pickup date selection.

1. Open pickup date selector.
2. Choose a valid date.
3. Confirm before submitting.

If pickup date is required and not selected, order/cart action may fail.

## 7. Cart Management

### 7.1 Add Item to Cart

1. Open a product.
2. Select size.
3. Set quantity.
4. Select pickup date if required.
5. Click Add to Cart.

### 7.2 View Cart

In Cart page, verify:

- Correct product
- Correct size
- Correct quantity
- Pricing summary

### 7.3 Update Quantity in Cart

1. Click quantity controls.
2. Increase or decrease as needed.
3. Wait for cart refresh/confirmation.

### 7.4 Remove Item from Cart

1. Click remove/delete icon on the item.
2. Confirm removal.
3. Verify item is removed from list.

## 8. Placing an Order

### 8.1 Standard Order Steps

1. Review cart items.
2. Confirm sizes and quantities.
3. Ensure pickup date is set when required.
4. Submit order.
5. Wait for success message.

### 8.2 Immediate Buy Flow

If Buy Now is available:

1. Open product.
2. Select required options.
3. Click Buy.
4. Confirm order details.

### 8.3 After Order Submission

Go to Orders page and monitor status transitions.

## 9. Order Tracking and Status Guide

### 9.1 Where to Track

Use /orders to view all your order records.

### 9.2 Status Meanings

- Pending: order was submitted and is awaiting processing.
- Ready for Pickup: order can now be claimed by customer.
- Declined: order was not approved.
- Completed: order flow is finalized.

### 9.3 What to Check Per Order

- Order ID
- Product name
- Quantity
- Size
- Pickup date
- Remarks or decline reason if applicable

## 10. Pickup Confirmation Process

When order is marked Ready for Pickup:

1. Open order details.
2. Verify product and quantity.
3. Click Confirm Pickup.
4. Enter OR number if requested.
5. Submit confirmation.
6. Verify status changes to Completed.

Important:

- Confirm pickup only when item is actually received.
- Once completed, rating action may become available.

## 11. Ratings and Reviews

### 11.1 When You Can Rate

Only completed orders are eligible for rating.

### 11.2 How to Submit Rating

1. Open Completed tab in Orders.
2. Click Rate Order.
3. Select 1 to 5 stars.
4. Add optional written review.
5. Submit.

### 11.3 Rating Rules

- Rating value must be from 1 to 5.
- One rating per order per user.
- Duplicate submission for same order is blocked.

### 11.4 Review Writing Tips

- Keep feedback clear and specific.
- Mention fit, quality, and overall experience.
- Avoid personal or sensitive data.

## 12. Messaging and Chat

### 12.1 Open Messages

1. Go to /messages.
2. Open an existing conversation or start interaction.

### 12.2 Send Message

1. Type your message.
2. Click send.
3. Confirm message appears in conversation history.

### 12.3 Receive Realtime Reply

When realtime connection is active, incoming messages appear without full page reload.

### 12.4 Messaging Best Practices

- Be concise and include order ID when asking support.
- Send one clear concern per message.
- Avoid sending duplicate messages repeatedly.

## 13. Notifications and Unread Indicators

Unread indicators help identify new messages or updates.

### 13.1 How to Clear Unread Items

1. Open the related message thread.
2. Read the latest messages.
3. Indicator should update automatically.

If indicator does not clear, refresh page and reopen conversation.

## 14. Common Errors and Troubleshooting

### 14.1 Cannot Log In

Possible causes:

- Wrong credentials
- Expired session/token
- Backend not reachable

Actions:

1. Re-enter credentials carefully.
2. Clear browser storage and login again.
3. Check internet connection.

### 14.2 Cannot Add to Cart

Possible causes:

- Size not selected
- Invalid quantity
- Session expired

Actions:

1. Select size first.
2. Ensure quantity is valid.
3. Re-login and retry.

### 14.3 Orders Not Showing

Possible causes:

- Request failed due to network/API issue
- Session expired

Actions:

1. Refresh the Orders page.
2. Re-login.
3. Retry after a short interval.

### 14.4 Cannot Confirm Pickup

Possible causes:

- Order is not yet in Ready for Pickup
- Required confirmation field (for example OR number) is missing

Actions:

1. Check current order status.
2. Complete required fields.
3. Submit again.

### 14.5 Cannot Submit Rating

Possible causes:

- Order is not completed
- Rating already exists for that order

Actions:

1. Confirm order is in Completed tab.
2. Check if the order has already been rated.

### 14.6 Messaging Not Realtime

Possible causes:

- Temporary socket connection issue
- Unstable internet connection

Actions:

1. Refresh Messages page.
2. Verify internet stability.
3. Retry after reconnection.

## 15. Security and Privacy Best Practices

- Use strong passwords with mixed characters.
- Do not share your credentials.
- Log out on public/shared devices.
- Avoid storing passwords in public browsers.
- Do not share sensitive personal details in chat.

## 16. Frequently Asked Questions

### 16.1 Do I need an account to order?

Yes. Protected actions such as cart, orders, ratings, and messages require login.

### 16.2 Why do I need to choose size first?

LocalFit validates orders per size. You must select a size before placing an order.

### 16.3 Can I rate an order more than once?

No. Each completed order can only be rated once by the same user.

### 16.4 Why is my pickup confirmation not working?

Most common reasons are incorrect order status or missing required fields.

### 16.5 What should I include when messaging support?

Include your order ID, concern summary, and specific issue details.

## 17. Glossary

- Cart: temporary list of selected products before ordering.
- Order Status: current processing stage of an order.
- Ready for Pickup: status indicating your order is available for claiming.
- Completed: status indicating order has been finalized.
- Rating: star-based feedback (1 to 5) for a completed order.
- OR Number: reference number that may be required during pickup confirmation.
